# Chatbot SVG Galerie

Dieses Projekt erstellt eine Galerie von SVG-Grafiken für Chatbots. Jede Grafik zeigt ein Icon in einem achteckigen Rahmen auf schwarzem Hintergrund.

## Funktionen

*   Generierung von SVG-Grafiken basierend auf einer Liste von Chatbot-Ideen.
*   Anzeige der Grafiken in einer responsiven HTML-Galerie.
*   Bereitstellung des Einbindungscodes für jede SVG-Grafik.

## Verwendung

1.  Öffnen Sie die `index.html`-Datei in Ihrem Webbrowser.
2.  Die Galerie zeigt die generierten SVG-Grafiken an.
3.  Klicken Sie auf die Schaltfläche "Kopieren" unter jeder Grafik, um den Einbindungscode in die Zwischenablage zu kopieren.

## Technologien

*   HTML
*   CSS
*   JavaScript
*   SVG
*   Markdown (für die Chatbot-Ideen-Liste)

## Projektstruktur

*   `index.html`: Die HTML-Datei für die Galerie.
*   `script.js`: Das JavaScript-Skript, das die SVGs generiert und die Galerie aktualisiert.
*   `chatbot-ideen.md`: Die Markdown-Datei mit den Chatbot-Ideen.
*   `chatbot-ideen-erweitert.md`: Die erweiterte Markdown-Datei mit zusätzlichen Chatbot-Ideen.
*   `README.md`: Diese Datei.
